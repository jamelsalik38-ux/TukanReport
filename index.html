Stack (Linked List-Based Implementation)

A Stack is a linear data structure that follows the LIFO (Last In, First Out) principle. The element inserted last is the first one to be removed.

Linked List-Based Implementation

Each node in a linked list-based stack holds data and a pointer to the next node. The top of the stack points to the most recently added node.

Advantages:

  • Dynamic size – no predefined limit
  • Efficient memory usage
  • No overflow unless system memory is full
# Linked List Stack Example
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class Stack:
    def __init__(self):
        self.top = None

    def push(self, data):
        node = Node(data)
        node.next = self.top
        self.top = node

    def pop(self):
        if not self.top:
            return None
        temp = self.top.data
        self.top = self.top.next
        return temp

stack = Stack()
stack.push(10)
stack.push(20)
print(stack.pop())  # Output: 20

Push Operation

Push adds an element on top of the stack.

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

# Example
s = Stack()
s.push(5)
s.push(10)
print(s.items)  # [5, 10]

Pop Operation

Pop removes the topmost element from the stack.

class Stack:
    def __init__(self):
        self.items = []

    def pop(self):
        if not self.items:
            return None
        return self.items.pop()

# Example
s = Stack()
s.push(5)
s.push(10)
print(s.pop())  # 10

Peek / Top Operation

Peek returns the top element without removing it.

class Stack:
    def __init__(self):
        self.items = []

    def peek(self):
        if not self.items:
            return None
        return self.items[-1]

# Example
s = Stack()
s.push(10)
s.push(20)
print(s.peek())  # 20

isEmpty and isFull

isEmpty() checks if the stack is empty, and isFull() is mainly used for fixed-size stacks.

class Stack:
    def __init__(self, capacity=None):
        self.items = []
        self.capacity = capacity

    def isEmpty(self):
        return len(self.items) == 0

    def isFull(self):
        return self.capacity and len(self.items) >= self.capacity

# Example
s = Stack(3)
print(s.isEmpty())  # True
s.push(1)
s.push(2)
s.push(3)
print(s.isFull())   # True

Applications of Stack

  • Expression Evaluation – Solving postfix or prefix expressions
  • Backtracking – Undo operations like in maze solving
  • Function Calls – Used to maintain call stack during recursion
# Example: Postfix Expression Evaluation

def evaluate_postfix(expr):
    stack = []
    for token in expr.split():
        if token.isdigit():
            stack.append(int(token))
        else:
            b, a = stack.pop(), stack.pop()
            if token == '+': stack.append(a + b)
            elif token == '-': stack.append(a - b)
            elif token == '*': stack.append(a * b)
            elif token == '/': stack.append(a / b)
    return stack[0]

print(evaluate_postfix("3 4 + 2 *"))  # Output: 14
